# Meter.by

Сайт для подключения к системе дистанционного съема показаний счетчиков холодной и горячей воды, а также тепла в Минске, Республика Беларусь.

## Описание

Meter.by — это веб-сайт компании, предоставляющей услуги по:
- Поверке счетчиков воды и тепла
- Монтажу и установке счетчиков
- Подключению к системе дистанционного съема показаний
- Обслуживанию и консультациям

Сайт включает в себя:
- Главную страницу с информацией о услугах
- Страницу с ценами и пакетами услуг
- Каталог документов и сертификатов
- Новостной блог
- Видеогалерею
- Форму генерации коммерческих предложений

## Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация с поддержкой темной темы и glassmorphism эффектов
- **JavaScript (Vanilla)** — интерактивность и функциональность
- **Bootstrap 5.3** — адаптивная сетка и компоненты
- **Node.js** или **Python** — локальный сервер для разработки

## Установка и запуск

### Вариант 1: Node.js сервер

1. Убедитесь, что у вас установлен Node.js
2. Запустите сервер:
```bash
node server.js
```
3. Откройте браузер по адресу: `http://localhost:8000`

### Вариант 2: Python сервер

1. Убедитесь, что у вас установлен Python 3
2. Запустите сервер:
```bash
python server.py
```
или
```bash
python3 server.py
```
3. Откройте браузер по адресу: `http://localhost:8000`

### Вариант 3: Windows (автоматический запуск)

Дважды кликните на файл `start-server.bat` — сервер запустится автоматически.

## Основные функции

- **Дистанционный съем показаний** — информация о подключении к системе
- **Поверка счетчиков** — услуги по поверке счетчиков воды и тепла
- **Монтаж и установка** — профессиональная установка оборудования
- **Генератор коммерческих предложений** — создание PDF предложений
- **Корзина услуг** — выбор и оформление пакетов услуг
- **Адаптивный дизайн** — поддержка мобильных устройств
- **Темная тема** — переключение между светлой и темной темой

## Разработка

Для разработки рекомендуется использовать локальный сервер (Node.js или Python), так как некоторые функции требуют работы через HTTP протокол.

### Добавление новых услуг

Редактируйте файл `data/services.json` для добавления или изменения услуг.

### Добавление новостей

Создайте новый HTML файл в папке `news/articles/` по шаблону `_template.html` и добавьте метаданные в `news/data/articles-metadata.json`.

### Изменение цен

Обновите файл `data/prices.json` с актуальными ценами и пакетами услуг.
