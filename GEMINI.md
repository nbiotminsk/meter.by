# Обзор проекта: Meter.by

Это проект статического веб-сайта для компании "Meter.by", которая предоставляет услуги по дистанционному съему показаний приборов учета в Республике Беларусь. Сайт создан с использованием современных веб-технологий и ориентирован на предоставление информации об услугах, ценах, а также содержит новостной раздел и галерею.

## Ключевые технологии

*   **Фреймворки и библиотеки:**
    *   **Bootstrap 5.3:** для адаптивной верстки и стилизации.
    *   **Alpine.js:** для создания интерактивных компонентов.
    *   **VanillaTilt.js:** для создания 3D-эффектов при наведении на карточки услуг.
    *   **SweetAlert2:** для отображения модальных окон.
    *   **html2pdf.js:** для генерации PDF-документов (коммерческих предложений).
*   **Данные:**
    *   Контент (услуги, документы, новости, видео) загружается динамически из JSON-файлов, расположенных в директории `data/`.
*   **Сборка и запуск:**
    *   Проект не требует сборки.
    *   Для локальной разработки используется простой HTTP-сервер на Python или Node.js.

## Структура проекта

*   `index.html`: главная страница.
*   `prices.html`, `documents.html`, `videos.html`, `news/index.html`: другие страницы сайта.
*   `css/`: стили, включая основной `style.css`, темную тему `dark-theme.css` и стили для glassmorphism-эффектов.
*   `js/`: скрипты, включая `main.js` (основная логика), `navigation-manager.js`, `image-optimization.js` и другие.
*   `data/`: JSON-файлы с данными для динамической загрузки контента.
*   `documents/`: PDF-документы (сертификаты, инструкции и т.д.).
*   `images/`: изображения, используемые на сайте.
*   `server.py`, `server.js`: простые серверы для локальной разработки.
*   `start-server.bat`: скрипт для запуска локального сервера.

## Запуск проекта

Для запуска проекта локально, используйте один из следующих способов:

1.  **С помощью `start-server.bat`:**
    *   Запустите файл `start-server.bat`. Он попытается запустить `server.py`. Для этого в системе должен быть установлен Python.

2.  **С помощью Python (рекомендуемый):**
    *   Убедитесь, что у вас установлен Python 3.
    *   В терминале выполните команду:
        ```bash
        python server.py
        ```
    *   Если команда `python` не работает, попробуйте `py server.py`.

3.  **С помощью Node.js:**
    *   Убедитесь, что у вас установлен Node.js.
    *   В терминале выполните команду:
        ```bash
        node server.js
        ```

После запуска сервера сайт будет доступен по адресу `http://localhost:8000`.

## Соглашения по разработке

*   **Динамический контент:** Весь основной контент (услуги, документы, новости, видео) должен добавляться и редактироваться через соответствующие JSON-файлы в директории `data/`. Это позволяет изменять контент без необходимости вносить правки в HTML-код.
*   **Изображения:** Для оптимизации производительности используется ленивая загрузка изображений, а также WebP формат.
*   **Интерактивность:** Для создания интерактивных элементов используется Alpine.js.
*   **Стили:** Стили разделены на несколько файлов для лучшей организации. Основные стили находятся в `css/style.css`.
